Interactive 3D Mesh Label Color Modification Tool

Step-by-step demonstration & Project Overview

• Developed a 3D mesh labeling tool supporting batch label modification, color mapping, undo, and saving, improving model processing efficiency.

• Implemented with Python using OpenMesh, Polyscope, and libigl for mesh reading, adjacency computation, and interactive visualization.

• Applicable to multiple .obj mesh files with interactive response time < 0.6s.

• Users can select different label colors via buttons and apply them by clicking mesh faces.

• “Edit Mode” toggle is provided; label modifications require enabling this option.

• Supports k-neighborhood selection: users can select a face and its k adjacent layers for batch modification.

Step 1

The user can see a 3D mesh model displayed in the Polyscope visualization interface.

Step 2

The user can interactively select label colors and modify label values by clicking buttons and dragging sliders.

Step 3

As the user interacts, the face colors of the mesh model are updated in real time to reflect the new label colors.

Step 4

The user can save the modified label information to a new TXT file and reload it when needed.
